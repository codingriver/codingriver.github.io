# 标签

当前站点采用“先写标签元数据，再逐步增强展示层”的策略。

## 当前使用方式

在文章头部增加 front matter：

```markdown
---
title: 示例文章
tags:
  - Shader
  - Unity
  - 图形学
categories:
  - shader
comments: true
---
```

## 现阶段约定

- 标签先写入文章元数据
- 标签页先手工维护
- 后续如需自动聚合，再评估插件化方案
