当前站点主要分为以下几个方向：

- `u3d`：Unity 开发、性能优化、UI、平台适配
- `shader`：Shader 基础、光照模型、特效实现
- `csharp`：C# 语法、异步、运行机制、工程实践
- `ANote`：项目记录、随笔、开发笔记
- `Android` / `IOS`：移动平台接入、跳转、审核、配置
- `math`：图形学和数学基础
