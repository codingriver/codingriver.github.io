## JetBrains之全家桶破解（idea,rider....）

[https://ckey.run/](https://ckey.run/)
 激活：
    Windows：复制命令 irm ckey.run|iex使用Win + X按键，选择WindowsPowerShell(管理员)运行即可全自动激活!
    Linux：复制命令 wget -q ckey.run -O ckey.run && bash ckey.run使用终端执行即可全自动激活！
    Mac：复制命令 curl -Ls ckey.run -o ckey.run && bash ckey.run使用终端执行即可全自动激活！

## Java环境安装

1. jdk的下载可以直接在官网进行，链接为<http://www.oracle.com/technetwork/java/javase/downloads/index.html>
至于版本下载8还是9请自行决定，推荐8版本
![](../images/2024-02-28-19-49-34.png)


2. 添加环境变量


![](../images/2024-02-28-19-54-28.png)
![](../images/2024-02-28-19-55-57.png)
![](../images/2024-02-28-19-57-33.png)

```
JAVA_HOME
C:\Program Files\Java\jdk1.8.0_121

CLASSPATH
%JAVA_HOME%\lib;%JAVA_HOME%\lib\tool.jar

在变量Path中追加
%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin
```
