# Sourcetree-Mac安装跳过注册登陆和选择服务器

1. 安装sourcetree
2. 关闭sourcetree
3. 命令终端输入```defaults write com.torusknot.SourceTreeNotMAS completedWelcomeWizardVersion 3```
4. 打开sourcetree即可跳过登录
